# equipments

装備品データを種類ごとに格納したディレクトリです。各サブディレクトリ名は装備タイプを表す ID プレフィックスとなっており、その中に JSON 形式のデータファイルが含まれます。

- `SMAA` — 対空砲 (anti-aircraft gun)
- `SMDC` — 指揮装置・火器管制装置
- `SMH` — 船体装甲パーツ
- `SMHG` — 大口径主砲
- `SMHR` — ロケット砲
- `SMLG` — 中口径主砲
- `SMLR` — 長距離ロケット砲
- `SMMG` — 機関砲
- `SMMN` — 船載ミサイル
- `SMRF` — 副砲

その他のサブディレクトリも同様の形式で装備データを収録しています。
